ProTanki LVK v1.2
by Aivaz
-----------------------------
Если у вас x64 Windows, то для запуска вам следует открыть файл ProTLVK
-----------------------------
Если у вас x32 Windows, то для запуска нужно открыть FlashPlayer и в него перетащить файл TanksTestingTool, если загрузка не идёт, то добавьте ProTLVK в доверенную папку через GlobalSettings
-----------------------------
Cтатья на ProTanki Wiki: https://wiki.playprotanki.com/ru/ProTLVK
-----------------------------
Управление в ProTLVK

~ (ё) -- открыть настройки
WASDQE -- управление камерой
Стрелочки и ZXC - управление танком
Shift -- ускорить скорость передвижения камеры
Shift-I -- замедлить/увеличить скорость передвижения камеры
Ctrl-Shift-K -- открыть консоль
Ctrl-0 -- сохранить позицию камеры
0 -- загрузить позицию камеры
G -- следующий тип респаунов
Shift-G -- предыдущий тип респаунов
T -- следующая точка респауна в текущей группе
Shift-T -- предыдущая точка респауна в текущей группе
Insert -- добавить танк
Ctrl-Insert -- добавить танки по всем точкам респауна в текущей группе
Delete -- удалить текущий танк
N -- следующий танк
Shift-N -- предыдущий танк
Клик по танку -- выбрать этот танк
F -- переключить режим камеры
Numpad_7/Numpad_9 -- сменить башню
Numpad_4/Numpad_6 -- сменить корпус
Numpad_1/Numpad_3 -- сменить краску
F12 -- сменить башню, копрус и краску на случайное
Ctrl-M -- загрузить собственную краску для текущего танка
Shift-R -- открыть панель получения звания для текущего танка
Shift-F -- открыть настройки user title(хп, нара, команда) для текущего танка
F2 -- открыть настройки верхней панели
F3 -- открыть редактор параметров пушки
F4 -- открыть редактор физики корпуса для текущего танка
F6 -- открыть редактор фейк статистики
Numpad-/Numpad+ -- изменить размер окна
[ -- предыдущее оружие
] -- следующее оружие
; -- предыдущий эффект оружия
' (э) -- следующий эффект оружия
1 -- сбросить аптеку
2 -- сбросить защиту
3 -- сбросить двойной урон
4 -- сбросить ускорение
5 -- сбросить кристалл
6 -- сбросить голд
Shift-6 -- сбросить голд с задержкой и сиреной
U -- сбросить голд в том месте, где находится курсор
I -- взять последний сброшенный ящик
О -- удалить последний сброшенный ящик
K -- перевернуться
L -- взорваться
J -- прыгнуть
M -- поставить мину
V -- сбросить флаг
R -- переместить текущий танк на текущую точку респауна и восстановить его нормальную ориентацию
H -- танк в фантом
Y -- телепортировать танк в то место, где находится курсор
Ctrl-F8 -- сохранить позицию танка
F8 -- телепортировать танк в сохранённую позицию
Ctrl-B -- включить/выключить отладочный режим камеры
F5 -- показывает/скрывает точки появления
Tab -- переключение режима отображения карты
Ctrl-G -- открыть гараж
Ctrl-Delete -- сбросить всё до заводских настроек
